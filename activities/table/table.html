When do you use a table?

Tables are used to display data in a grid format. They are used when you have data that is best displayed in rows and columns. Tables are not used for layout purposes.

When do you not use a table?

Tables are not used for layout purposes. They are not used to create columns on a webpage. Tables are not used to create a grid layout for a webpage.

When do you need to do extra work for table accessibility?

Extra work is needed for table accessibility when you have a table that spans multiple pages. You need to add a summary attribute to the table tag to describe the table. You also need to add headers to the table cells to make the table accessible to screen readers.
