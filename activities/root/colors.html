How do you select an item?

by using an ID or a class such as #id or .class

What's the difference between an ID and a Class?

ID's are unique and should only be used for one element. Classes can be used for multiple elements.

What's the difference between a span tag and a div tag?

span is an inline element used for one thing, and div is a block element used for many things

What's the difference between a DIV tag and a Header, Main or Footer?

DIV is a block element used for many things, and Header, Main, and Footer are semantic elements that describe their purpose

How do you select colors?

by using a color name, a hex code, or an RGB code
