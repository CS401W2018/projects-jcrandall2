Internet

The internet is a bunch of computers that communicate with each other on a global network.

Web

The web is a bunch of web pages connected together

How are the Internet and the Web different?

The internet is computers communicating with each other and the web is pages connected together.

ASCII and Unicode

ASCII is something everyone uses to encode characters on computers and other machines that use ASCII. It uses a 7 bit binary language which means any combination of zeros and ones in 7 character spots code for 127 different letters, numbers or commands. Unicode is similar but it uses 16 bit binary and includes characters from outside the English language.

UTF-8

UTF-8 is like ASCII but uses 8 bit binary. It is the most commonly used coding standard on the web.

URL

URL is short for Uniform Resource Locator. It's basically the address of a webpage.

Domain Name

A domain name is what the internet user types in to get to a website i.e. typing in facebook.com to get to facebook.

HTTP and HTTPS

HTTP is short for Hypertext Transfer Protocol. It's used transfer data between the server and browser. HTTPS uses encryption and is just safer than HTTP

IP Address

An IP address is an address made up of numbers and periods used to identify the network a computer is using. It's good to track the location of where someone is using a computer

DNS

DNS is short for Domain Name System. It changes a domain name into an IP address so other computers can interact with it

W3C Recommendations

The World Wide Web Consortium (W3C) are basically the government of the internet. They come up with rules and standards to help the internet run smoothly around the world. They recommend using certain things to make websites so the internet itself is more uniform and effiecient

XML, XHTML, HTML, HTML5

XML is short for Extensible Markup Language. It makes rules for things such as documents to be readable by both human and machine when they're encoded. XHTML is short for Extensible Hypertext Markup Language. It just makes HTML into XML format. HTML is short for Hypertext Markup Language. It is the standard for making websites. HTML5 is the latest version of HTML. It has a bunch of new features like videos and canvas.
