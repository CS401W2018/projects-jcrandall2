Internet

The internet is a bunch of computers that communicate with each other on a global network.

Web

The web is a bunch of web pages connected together

How are the Internet and the Web different?

The internet is computers communicating with each other and the web is pages connected together.

ASCII and Unicode

ASCII is a character encoding standard that uses 7-bit binary numbers to represent text characters. Unicode is a character encoding standard that uses 16-bit binary numbers to represent text characters.

UTF-8

UTF-8 is a character encoding standard that uses 8-bit binary numbers to represent text characters. It is the most widely used character encoding standard on the web.

URL

URL stands for Uniform Resource Locator. It is the address of a web page on the internet.

Domain Name

A domain name is the human-readable address of a website, such as google.com or facebook.com.

HTTP and HTTPS

HTTP stands for Hypertext Transfer Protocol, and it is the protocol used to transfer data between a web server and a web browser. HTTPS is a secure version of HTTP that uses encryption to protect data.

IP Address

An IP address is a unique numerical address that identifies a device on a network, such as the internet.

DNS

DNS stands for Domain Name System. It is the system that translates domain names into IP addresses so that computers can communicate with each other over the internet.

W3C Recommendations

The World Wide Web Consortium (W3C) is an international community that develops open standards to ensure the long-term growth of the web. W3C Recommendations are guidelines and specifications that web developers should follow to create accessible and interoperable web content.

XML, XHTML, HTML, HTML5

XML stands for Extensible Markup Language, and it is a markup language that defines rules for encoding documents in a format that is both human-readable and machine-readable. XHTML stands for Extensible Hypertext Markup Language, and it is a reformulation of HTML as an XML application. HTML stands for Hypertext Markup Language, and it is the standard markup language used to create web pages. HTML5 is the latest version of HTML, and it includes new features such as video, audio, and canvas elements.
